### - 1️⃣ What is the difference between var, let, and const?
answer:
      var => var is a function-scoped variable, it is not block scoped. The same variable can be re-declared and its value can be changed. Also, var is hoisted, meaning it is initialized to undefined before the code runs.
       
       let => let is a block-scoped variable, it cannot be re-declared in the same scope, but its value can be changed.
      
       const => const is block-scoped, hoisted but cannot be accessed before declaration Temporal Dead Zone, and its value cannot be reassigned after initialization
